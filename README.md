# Minecraft 1.8.9 FML Mod 开发教程

欢迎来到本教程的源代码仓库！

由于相关代码严重过时，本教程于2020年11月4日正式封存。

如果对教程有疑问，可以通过电邮的方式联系本人：[zzzz@infstudio.net](mailto:zzzz@infstudio.net)。

教程所属的代码仓库的`master`分支是根据所有维护的patch自动生成的，教程的文字部分由`book`分支维护，代码部分由`patch`分支维护。
